# bayes-ultimate-slash-command

AI Agent Okinawaで紹介したプロンプトです。
以下形式でプロンプトを渡すと動作します
```
/issue-worker [やってほしいこと] #[issue番号(省略可能)]
```

最初のプロンプトは以下がおすすめ

*すでにプロジェクトが存在している場合*
```
/issue-worker このプロジェクトのspecを作成してください。
```

*新規にプロジェクトを立ち上げる場合場合(任意で修正してください)*
```
/issue-worker [プロジェクトの背景･目的: xx][実装言語: xx][その他詳細情報: xx]まずspecを作成してください。
```
